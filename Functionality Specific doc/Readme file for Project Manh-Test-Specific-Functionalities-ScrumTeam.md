Readme File - Project Manh-Test-Specific-Functionalities-ScrumTeam

Below are the parameters configured for this job
 

1.	Environment- This dropdown allows you to select the environment where you want to run the scripts. User can select any 1 environment from the below list (DEV, TEST, STAGE)
 

2.	Customer- This dropdown allows you to select the customer for which you want to run the scripts. User can select any 1 customer from the drop down and depending on the customer selected the functionalities of that customer will be loaded automatically.

 

3.	Functionalities- This field allows the user to select the functionalities they want to execute for the customer selected in above step.
User can select multiple functionalities at once by selecting multiple check boxes
 

4.	Execution Platform- This dropdown allows you to select the platform where you want to run the scripts. User can select either OCI Windows chrome or OCI POD form the dropdown.
 

5.	Select PODIP- This dropdown allows you to select the POD ID where you want to run the scripts. This dropdown is only applicable if user has selected OCI POD in execution platform above.

 

6.	Mail_to- This text box allows user to input their Email address. Whenever the job will be executed completely, a HTML report will be sent to the given Email address.
